# Household Transaction Analysis (2015-2018)

## Overview
This project provides a deep dive into three years of personal financial data. It focuses on cleaning messy transaction logs and uncovering hidden patterns in daily spending habits.

## Key Insights
- **Top Categories:** Food and Transportation lead in frequency.
- **Skewed Spending:** Most transactions are under 100 INR, but occasional large expenditures significantly shift the average.
- **Seasonality:** Time-series analysis revealed clear monthly peaks, suggesting seasonal financial cycles.
